# 功能修改

1. 修改 yt-dlp 下載字幕 
例如 https://www.youtube.com/watch?v=X9t-u87df3o 
使用 v= { id } 使用 id 建立資料夾。資料夾目錄下 包含  en.srt 字幕。封面照片。 聲音檔 mp3 等資訊。
2. sqlite 建立資料表格 videos 。將資料夾的 內的連結 放入 videos 表格。
